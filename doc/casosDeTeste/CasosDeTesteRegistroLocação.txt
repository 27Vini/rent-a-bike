Caso 1 : 
    Cenário: Cadastro realizado com sucesso
    Dado que desejo salvar meus dados
    quando clico em cadastrar
    então deve mostrar mensagem de sucesso

Caso 2:
    Cenário: Dado não preenchido
    Dado que não preenchi um campo
    quando clico em cadastrar
    então deve mostrar mensagem de erro

Caso 3:
    Cenário: Pesquisar cliente retorna cliente correto
    Dado que desejo pesquisar um cliente
    quando digito seu código ou CPF
    então deve mostrar aquele cliente corretamente

Caso 4:

    Dado que desejo pesquisar um cliente inexiste

Caso 5:

Caso 6:

Caso 7:

    Dado que escolhi um item correto

Caso 8:
